# AOF — Sales Deck

Vertriebspräsentation für Außendienstmitarbeiter.

**Nicht öffentlich** — nur für internen Gebrauch.

## Dateien

- `index.html` — Die komplette Präsentation (Single-File, 424 KB)
- `404.html` — Fehlerseite
- `vercel.json` — Hosting-Config
- `robots.txt` — Blockiert Google-Indexierung

## Inhalt

26 Folien in vier Routen:

- **Cover + Rollenauswahl** (2 Folien)
- **Anbieter-Route** (9 Folien): für Händler und Marktbetreiber
- **Werber-Route** (8 Folien): für Marken und Hersteller
- **Booker-Route** (7 Folien): für Agenturen und Vermarkter

## Bedienung

- **Pfeiltasten / Leertaste:** weiter/zurück
- **H:** zurück zur Rollenauswahl
- **N:** Präsentator-Notizen ein/aus
- **F:** Vollbild-Modus
- **Esc:** Menü/Notizen schließen
- **Klick auf Logo:** zurück zur Rollenauswahl

## Änderungen

Wenn Inhalte angepasst werden müssen: neue `index.html` ins Repo pushen, Vercel deployt automatisch.
